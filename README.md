# 文件说明

- *.ipynb: 数据预处理的过程和模型参数选择的过程

- main(.*).py: 根据ipynb的结果而选出的模型

- tool: 封装有对模型评估(predict.py)和数据处理(preprocessing.py)的类

- algorithm: 训练模型的几个算法

- datasets: 数据集
